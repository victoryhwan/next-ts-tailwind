# MacOs에서 Jenkins 설치
_`Jenkis`는 가장 많이 사용하는 CI/CD Tool로 알려져 있다. 다양한 플러그인이 있어서 기술적으로 안되는게 거의 없다고 보면 된다._
## 명령어 모음
```shell:terminal
//1. 홈브루 업데이트
brew update

//2. 젠킨스 설치
brew install jenkins-lts

//3. 젠킨스 시작
brew services start jenkins-lts

//4. 하단 위치로 이동 후 비밀번호 복사
vi /Users/내컴퓨터위치/.jenkins/secrets/initialAdminPassword

//5. 하단 사이트에 비밀번호 입력
http://localhost:8080

//6. install suggested plugin 선택

//7. 사용자 만들고 시작
```

이제 필요한 플러그인을 설치하고 CI/CD에 필요한 환경을 설정하여 구성하면 된다.
